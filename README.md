📌 Overview
This project is a simple Flight Reservation System built with Python and Tkinter, using SQLite database for storing reservation details.
It demonstrates the implementation of CRUD operations:
Create – Book a new flight reservation.
Read – View a list of all reservations.
Update – Modify existing reservations.
Delete – Remove a reservation.
The app provides a user-friendly GUI to manage reservations efficiently.
⚙️ Requirements
Before running the project, make sure you have:
Python 3.10+ installed
Required dependencies (usually come with Python):
tkinter
sqlite3
🚀 How to Run
Clone the repository (or download as ZIP and extract):
git clone https://github.com/salmasamir0/Flight-Project.git
cd "Flight Project"
Run the main application:
python main.py
The GUI window will open, allowing you to:
Add new reservations
View all reservations
Update reservation details
Delete reservations
